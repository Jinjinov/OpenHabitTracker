用 Markdown 记笔记、规划任务、培养习惯 — 提供 Windows、Linux、Android、iOS、macOS 原生应用及网页版。免费、无广告、开源。无需账户,所有数据保存在你的设备上。

没有连续打卡压力的习惯养成

每个习惯都有一个期望的重复间隔,而不是连续打卡计数器。应用会以百分比显示每个习惯相对于间隔的逾期程度,让你一眼看出最需要关注的事。错过一天不会清零任何东西。经典的连续打卡统计也有 — 默认关闭。

笔记、任务、习惯,一个应用全搞定

• 完整支持 Markdown 的笔记
• 支持子项、优先级和计划日期的任务
• 支持灵活间隔、计时器、次数和数量的习惯
• 分类、搜索、筛选和排序

数据完全属于你

• 所有数据本地保存在你的设备上
• 导出/导入:Markdown、JSON、YAML、TSV
• 从 Google Keep 导入笔记
• 可选同步:在自己的服务器上运行免费 Docker 镜像,桌面和移动端应用都能登录同步

随你定制

• 26 款主题,支持深色和浅色模式
• 支持 20 种语言
• 支持键盘操作和屏幕阅读器
